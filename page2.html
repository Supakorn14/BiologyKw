สูตรพันธุศาสตร์

หน้ารวบรวมสูตรและตัวอย่างเกี่ยวกับพันธุศาสตร์ เพื่อช่วยให้คุณเข้าใจและคำนวณได้ง่ายขึ้น

1. สูตรพื้นฐาน

  • ความน่าจะเป็น: P = จำนวนลักษณะที่ต้องการ / จำนวนลูกทั้งหมด
  • โมโนไฮบริด: ใช้ Punnett Square แสดงโอกาสลักษณะเด่น/ด้อย
  • ไดไฮบริด: โอกาสลักษณะรวม = ผลคูณของโอกาสแต่ละคู่

2. ตัวอย่างการสร้างตารางพันธุกรรม

2.1 ตารางพันธุกรรมแบบโมโนไฮบริด

ตัวอย่าง: Aa x Aa

A (พ่อ) a (พ่อ)
A (แม่) AA Aa
a (แม่) Aa aa

โอกาสลักษณะเด่น (AA, Aa): 3/4

โอกาสลักษณะด้อย (aa): 1/4

2.2 ตารางพันธุกรรมแบบไดไฮบริด

ตัวอย่าง: AaBb x AaBb

โอกาสลูก AABb: 1/4 x 1/2 = 1/8

3. สูตรความน่าจะเป็นแบบทวินาม

สูตร: P(X = k) = C(n, k) * p^k * (1-p)^(n-k)

  • C(n, k): การจัดหมู่ = n! / [k! * (n-k)!]
  • p: โอกาสลักษณะปรากฏ
  • n: จำนวนลูก
  • k: จำนวนลูกที่มีลักษณะเด่น

4. ตัวอย่างโจทย์

โจทย์: พ่อแม่ Aa x Aa, ลูก 4 คน อย่างน้อย 3 คนมีลักษณะเด่น

คำตอบ:

  • โอกาสลักษณะเด่น: 3/4
  • ใช้สูตร: P(X ≥ 3) = P(X = 3) + P(X = 4)
  • คำนวณ: P(X = 3) = C(4, 3) * (3/4)^3 * (1/4)^1 = 108/256
  • คำนวณ: P(X = 4) = C(4, 4) * (3/4)^4 * (1/4)^0 = 81/256
  • ผลรวม: 189/256 ≈ 73.83%
