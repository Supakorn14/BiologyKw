สูตรพันธุศาสตร์

หน้านี้รวบรวมสูตรและข้อมูลสำคัญเกี่ยวกับพันธุศาสตร์สำหรับการคำนวณความน่าจะเป็นและการวิเคราะห์ทางพันธุกรรม

1. สูตรพื้นฐาน

  • ความน่าจะเป็นของลักษณะเด่น/ด้อย: P = จำนวนลักษณะที่ต้องการ / จำนวนลูกทั้งหมด
  • การคำนวณพันธุกรรมแบบโมโนไฮบริด (Monohybrid Cross): ใช้การสร้างตารางพันธุกรรม (Punnett Square) เพื่อแสดงโอกาสการแสดงลักษณะเด่นและด้อย
  • การคำนวณพันธุกรรมแบบไดไฮบริด (Dihybrid Cross): ความน่าจะเป็นของลักษณะรวม = ผลคูณของลักษณะแต่ละคู่

2. ตัวอย่างการสร้างตารางพันธุกรรม

2.1 ตารางพันธุกรรมแบบโมโนไฮบริด

ตัวอย่าง: Aa x Aa

A (พ่อ) a (พ่อ)
A (แม่) AA Aa
a (แม่) Aa aa

ผลลัพธ์:

  • ความน่าจะเป็นลักษณะเด่น (AA, Aa): 3/4
  • ความน่าจะเป็นลักษณะด้อย (aa): 1/4

2.2 ตารางพันธุกรรมแบบไดไฮบริด

ตัวอย่าง: AaBb x AaBb

ความน่าจะเป็นของลูกที่มีลักษณะ AABb:

  • ความน่าจะเป็นที่ได้ AA: 1/4
  • ความน่าจะเป็นที่ได้ Bb: 1/2
  • ผลรวม: 1/4 x 1/2 = 1/8

3. สูตรความน่าจะเป็นแบบทวินาม (Binomial Probability)

ใช้สำหรับการวิเคราะห์ความน่าจะเป็นของลูกในหลายกรณี เช่น โอกาสที่ลูก 3 คนจากทั้งหมด 4 คนมีลักษณะเด่น

สูตร: P(X = k) = C(n, k) * p^k * (1-p)^(n-k)

โดย:

  • C(n, k): การจัดหมู่ = n! / [k! * (n-k)!]
  • p: ความน่าจะเป็นที่ลักษณะปรากฏ
  • n: จำนวนลูกทั้งหมด
  • k: จำนวนลูกที่มีลักษณะนั้น

4. ตัวอย่างโจทย์

โจทย์: พ่อแม่พันธุกรรม Aa x Aa ต้องการทราบโอกาสที่ลูก 4 คน อย่างน้อย 3 คนจะแสดงลักษณะเด่น

เฉลย:

  • โอกาสที่ลูกแสดงลักษณะเด่น: 3/4
  • ใช้สูตร: P(X ≥ 3) = P(X = 3) + P(X = 4)
  • คำนวณ: P(X = 3) = C(4, 3) * (3/4)^3 * (1/4)^1 = 108/256
  • คำนวณ: P(X = 4) = C(4, 4) * (3/4)^4 * (1/4)^0 = 81/256
  • ผลรวม: P(X ≥ 3) = 189/256 ≈ 73.83%
